# MQTT IoT: Управление освещением теплицы

Система автоматически управляет LED-подсветкой в теплице на основе данных с датчика освещённости, используя MQTT для передачи данных.
## Видео
https://photos.app.goo.gl/bW9FBF6wD3BAYHCs5
## Архитектура
```
[Arduino с датчиком] → serial → [publisher.py] → MQTT → [recivier.py] → serial → [Arduino с LED]
```

## Файлы
- `publisher.ino` — читает аналоговое значение с пина A5 и отправляет его в Serial.
- `publisher.py` — получает значение из Serial и публикует его в MQTT-топик.
- `recivier.ino` — ожидает команды `u` (включить) или `d` (выключить) по Serial и управляет LED на пине 3.
- `recivier.py` — подписывается на MQTT-топик, получает значение освещённости и отправляет команду `u` или `d` в Arduino в зависимости от порога.

## Логика работы
- Датчик выдаёт значение от 0 (темно) до 1023 (светло).
- Если значение **больше 1020**, свет **включается** (`u`).
- Если значение **меньше или равно 1020**, свет **выключается** (`d`).
